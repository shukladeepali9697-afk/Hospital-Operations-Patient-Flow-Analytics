
# 🏥 Hospital Operations & Patient Flow Analytics Dashboard
Hospital Operations & Patient Flow Analytics Dashboard using Python, Pandas, NumPy, SQL (BigQuery) and Tableau.
## 📌 Project Overview

This project analyzes hospital operations and patient flow using Python, SQL, and Tableau. The dashboard helps monitor hospital performance, patient distribution, revenue, insurance providers, and admission trends.

---

## 🛠 Tools Used

- Python (Pandas, NumPy)
- SQL
- Tableau Public
- Excel

---

## 📊 Dashboard KPIs

- Total Revenue
- Total Patients
- Average Billing
- Average Age

---

## 📈 Dashboard Visualizations

- Patient Distribution by Medical Condition
- Revenue by Insurance Provider
- Top 10 Hospitals by Revenue
- Monthly Admissions Trend
- Admission Type Distribution
- Patients Distribution by Age Group
